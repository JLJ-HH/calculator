=========================================
STRUKTURPLAN: SmartCalc (Python/Tkinter)
=========================================

MAIN APP: TaschenrechnerGUI (Controller & View)
 ├─ Komponenten:
 │   ├─ display (Entry für I/O)
 │   ├─ display_var (Reaktive Textbindung)
 │   └─ button_frame (Grid-Layout für 42+ Buttons)
 ├─ Interaktions-Logik:
 │   ├─ tastatur(event) -> Keyboard-Support
 │   ├─ toggle_dark() -> UI-Themes
 │   ├─ calculate() -> Zentraler Prozess-Trigger
 │   └─ shortcut_overlay() -> User-Hilfe
 └─ Instanzen:
     ├─ parser (RechenParser)
     └─ verlauf (VerlaufManager)

-----------------------------------------

MODULE 1: RechenParser (Logic Engine)
 ├─ Security: Whitelist-Validierung (erlaubte_zeichen)
 └─ berechne(eingabe):
     ├─ String-Preprocessing (RegEx für sin, cos, √, ^)
     ├─ Konstanten-Mapping (π, e)
     └─ Error-Handling (Division durch 0, SyntaxError)

-----------------------------------------

MODULE 2: VerlaufManager (Data Management)
 ├─ Storage: RAM-basierte Liste (eintraege)
 ├─ speichern(eingabe, ergebnis): Append-Logik
 └─ anzeigen(master): Dynamisches Toplevel-Window
     └─ Text-Widget + Scrollbar (UI-Kapselung)

=========================================
